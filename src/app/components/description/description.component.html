説明

このサイトは、テキストをモールス信号に変換することを目的に作成されました。
モールス信号に関連した3つの機能を実装しています。

・テキスト → モールス信号

入力したテキストをモールス信号に変換する機能となります。
入力できるテキストは、ひらがな・カタカナ(濁点、半濁点にも対応)、半角英数字、一部の記号(半角、全角両方に対応)となります。
対応していない文字は翻訳されず、無視されます。
なお、本機能で変換されたモールス信号では長点(ツー)を"－"(全角の長音)、短点(トン)を”・”（全角の中点）、
文字の間隔を2連続の半角スペースで表しています。
例1: 「SOS」 ⇨ 「・・・ －－－ ・・・」
例2: 「ララランド」 ⇨ 「・・・ ・・・ ・・・ ・－・－・ ・・－・・ ・・」

・欧文モールス信号 → アルファベット

入力した欧文モールス信号をテキストに変換する機能となります。
モールス信号は長点(ツー)を"_"(半角のアンダーバー)、短点(トン)を”.”（半角のピリオド）、
文字の間隔をひとつの半角スペースとして、入力してください。
アルファベットと数字と一部記号の変換に対応していて、対応していない文字は翻訳されず、無視されます。
例1: 「... . _._ .. ._. ___」 ⇨ 「SEKIRO」
例2: 「.... ._ _ . .._. .._ ._.. ___..」 ⇨ 「HATEFUL8」

・和文モールス信号 → カタカナ

入力した和文モールス信号をテキストに変換する機能となります。
モールス信号は長点(ツー)を"_"(半角のアンダーバー)、短点(トン)を”.”（半角のピリオド）、
文字の間隔をひとつの半角スペースとして、入力してください。
カタカナと数字と一部記号の変換に対応していて、対応していない文字は翻訳されず、無視されます。
例1: 「.__ _.._ ..__ ___._ ___._ _..._」 ⇨ 「ヤマノススメ」
例2: 「.__. _.._ _._._ _._.. ._... .._.. __._. .._.. _._ _. __._.」 ⇨ 「ツマサキオトシトワタシ」
